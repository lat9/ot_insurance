INSTALLATION
  1. Patch your database by copying and pasting the contents of install.sql into ADMIN->TOOLS->INSTALL SQL PATCHES
  2. Upload all files to your server, maintaining directory structure
  3. Define your shipping zones in Zen Cart prior to installing this module
  4. Go to ADMIN->MODULES->ORDER TOTAL->SHIPPING INSURANCE and click INSTALL
  5. Configure the module and create table rates for each shipping zone.
TIPS

If you need to add a zone after this module is installed, you will need to save your configuration settings to a text file and reinstall the module through the admin.

  • Use Table Rates? Set this to true is you want to define different rates for each of your defined zones. Set to false to use the increment or percentage methods.
  • Alternate Insurance Type: Set to percent to return insurance rate as a percentage of the order total. Set to amount to use the increment method below.
  • Insurance Percentage: If Alternate Insurance Type is set to percent, enter the percentage of the order total to be added for insurance.
  • Insurance Rate: If Alternate Insurance Type is set to amount, enter the rate to charge per every X amount (defined in the next section) of the order total.
  • Increment Amount: Charge the Insurance Rate for every Increment Amount in the order total.
  • Amount Exempt From Fee: If the order total falls below this amount, insurance is not charged or is deemed as included in the shipping rate.
  • Tax Class: add appropriate tax to the returned insurance rate.
  • No Insurance Fee on Virtual Products: Set to true to not include insurance if all products in the cart are virtual.
  • No Insurance Fee on Gift Vouchers: Set to true to not include insurance if all products in the cart are gift vouchers.
  • No Insurance Fee on Free Shipping: Set to true to not include the prices of items with free shipping in the order total. The insurance rate returned will be calculated on only products that return a shipping rate. This will make it so that all virtual and gift vouchers are not calculated as part of the order total.
  • Required Insurance Amount: If the order total is above this amount, shipping insurance is required and is automatically calculated in the order total without the option to be able to remove it by the customer.
  • Table Rates: a rate table is created for each Zone you have defined in your admin. Please follow the example given in the configuration page for instructions on how to use the table rates.
UNINSTALL
  1. Delete all uploaded files from your store.
  2. Go to ADMIN->MODULES->ORDER TOTAL->SHIPPING INSURANCE and click REMOVE
